# Food Delivery Website

Це вебсайт для замовлення доставки їжі з ресторанів. Система допомагає користувачу швидко знайти заклади, переглянути меню, додати страви в кошик та оформити замовлення з доставкою.

## Яку проблему вирішує
- Економить час на пошук їжі та оформлення замовлення
- Дозволяє робити замовлення онлайн без дзвінків
- Дає можливість відстежувати статус замовлення та отримувати сповіщення

## Основні функції
- Реєстрація та вхід користувача
- Пошук ресторанів та фільтрація за кухнею/рейтингом/часом доставки
- Перегляд меню ресторану та деталей страв
- Кошик: додавання/видалення страв, зміна кількості
- Оформлення замовлення (адреса, час доставки, спосіб оплати)
- Статус замовлення (прийнято → готується → в дорозі → доставлено)
- Історія замовлень

## Для кого цей продукт
- Для людей, які хочуть швидко замовити їжу додому або на роботу
- Для студентів та працівників, які часто замовляють доставку
- Для користувачів, які хочуть мати зручний онлайн-сервіс замовлення їжі
