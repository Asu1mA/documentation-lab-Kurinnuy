# SETUP GUIDE — Інсталяція та запуск (макет)

Цей документ описує запуск демо-версії вебсайту доставки їжі.

## Необхідні інструменти
- Будь-який сучасний браузер: Chrome / Edge / Firefox
- Доступ до інтернету
- (Опційно) акаунт для входу (email + пароль)

## Запуск (варіант 1 — лише перегляд сайту)
1) Відкрити посилання на сайт (демо).
2) Переконатися, що головна сторінка завантажилась: є пошук і список ресторанів.

## Запуск (варіант 2 — локально, якщо це макет-проєкт)
1) Завантажити файли проєкту на комп’ютер (Download ZIP або git clone).
2) Відкрити папку проєкту.
3) Відкрити файл `index.html` у браузері подвійним кліком.
4) Якщо є локальний сервер:
   - встановити Node.js (за потреби)
   - виконати `npm install`
   - виконати `npm run dev`
   - відкрити адресу `http://localhost:3000`

## Перевірка роботи
Вважаємо, що система запущена правильно, якщо:
- Відкривається головна сторінка зі списком ресторанів
- Працює перехід на сторінку ресторану
- Можна додати страву в кошик і побачити її там
- Кнопка “Оформити замовлення” відкриває форму замовлення
